Calculator – NexusCalc

Description:

A smart, dual-mode calculator supporting both standard arithmetic and scientific operations. Includes functions like sin, cos, tan, square root, square, cube, reciprocal, logarithm, and parentheses. Features a clean dark-themed UI with responsive design for all devices.

Tech Stack: HTML, CSS, JavaScript, Font Awesome

Key Features:

Standard and Scientific modes

Trigonometric functions (sin, cos, tan)

Square, cube, square root, reciprocal, logarithm

Parentheses support

Toggle sign (+/-)

Keyboard-responsive buttons
